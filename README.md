# Batch 07 Assignment 01

### <ins>Assignment: Advanced Problem Solving with TypeScript & OOP</ins>

This assignment includes several problems that demonstrate core TypeScript concepts such as data types, interfaces, classes with inheritance, type checking, and data structure manipulation. The solutions are designed to present clear, efficient, and well-structured problem-solving approaches while following TypeScript best practices.

## This assignment contains seven problems:

- Filtering even numbers
- String reversing
- Type checking
- Getting specific property using key
- Toggling read status of an object
- Class inheritance and method creation
- Array intersection

## This assignment also includes two blogs:

_Q1: How do Generics allow you to build reusable components and functions that stay strictly typed regardless of the data structures passed in?_
_Q2: How do Pick and Omit utility types prevent code duplication while creating specialized "slices" of a master interface? Discuss how this keeps your code DRY (Don't Repeat Yourself)._

## Coding Requirements

- Function names are exactly as specified in the problem descriptions.
- Returned values match sample output including spaces, punctuation, and capitalization.
- Meaningful words are used in function names and variables
- No unnecessary comments
- No `console.log`. Values are returned instead
